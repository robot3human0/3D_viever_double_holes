💮 3D_Viewer документация: 💮

3D_Viewer приложение для просмотра 3D моделей в каркасном виде, просто и удобно.

Установка и запуск:

  • Для простой установки предусмотрен "мэйкфайл" с целью install
  • Для запуска есть цель run (по умолчанию цель run предусматривает выполнение цели install)

Интерфейс программы:

  1. Кнопки открытия и сворачивания окна

  2. Меню бар
    3. 2.1. About_this_App
    2.1.1 Documentation - открывает данную страницу
    2.1.2 Authors - открывает диалоговое окно с информацией об авторах
    2.2 Do_a...
    2.2.1 gif - делает гифку с 3D моделью
    2.2.1 image - делает скриншот 3D модели

  3. В правом углу мемная кнопка откроет окно проводника для выбора .obj фаила

  4. Слева от мемной кнопки содержится информация о выбранном файле

  5. Ниже расположилась область перемещения модели по осям
    6. Область содержит в себе 3 поля для ввода значений для перемещения обьекта в пространстве по осям и кнопку Translate для применения значений

  6. Область вращения модели
    7. Содержит поля для ввода значений для каждой оси и кнопку

  7. Область для задачи масштаба 3D модели в ручную
    8. Содержит поле для ввода значения и кнопку

  8. Область для тонкой настройки отображения вершин
    9. Содержит в себе поле для изменения размера вершин, кнопку Vert_color для изменения цвета вершин, а так же можно задать тип вершины с помощью кнопок None, Circle, Square

  9. Область для тонкой настройки отображения линий и ребер 3D модели
    10. Содержит поле для задания толщины линий, кнопку Edge color для изменения цвета линий, а также можно изменить вид линий с помощью кнопок Smooth и Stippled соответственно

  10. Область кастомизации проекции
    11. Содержит кнопки Orthogonal и Perspective, а так же кнопку для изменений фона Background color

Makefile:

  • all
    • Произведет установку программы

  • install
    • Так же выполнит установку программы (не обязательно ее прописывать можно просто напечатать make)

  • run
    • Запустит программу, если же программа не установлена, выполнит установку

  • uninstall
    • Удалит программу

  • test
    • Протестирует функции написанные на си

  • dvi
    • Откроет данную документацию

  • gcov_report
    • Соберет статистику по модульному тестированию и откроет в браузере результат

  • dist
    • Выполнит создание tar-файла содержащего полный установочный пакет программы 3D-Viewer

  • clean
    • Удалит все ненужные и временые файлы

  • style
    • Проверит написанную библиотеку на google стиль
